# v0.1.10

## 주요 변경

- 앱 업데이트와 배포 파일 업로드 대상을 `hyjoong/worktree-manager` Releases로 변경했습니다.
- 소스 저장소를 public으로 전환하면서 별도 release 전용 저장소 의존성을 줄였습니다.
- 기존 v0.1.9 앱이 업데이트 경로를 잃지 않도록 이번 버전은 기존 release 채널에도 함께 업로드합니다.
- README의 다운로드 링크와 배포 메모를 현재 release 채널 기준으로 정리했습니다.
- 업데이트 상태 UI를 우측 `App` 탭 중심으로 정리하고 상태 표시 테스트를 추가했습니다.
