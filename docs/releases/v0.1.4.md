# v0.1.4

## 주요 변경

- 앱 자동 업데이트 기반 추가
  - GitHub Releases 기반 업데이트 확인
  - packaged 앱 시작 시 자동 업데이트 확인
  - 업데이트 다운로드 완료 후 재시작 설치
- 우측 `App` 탭 추가
  - 현재 앱 버전 표시
  - development/packaged 모드 표시
  - 업데이트 상태, 최신 버전, 진행률 표시
  - 업데이트 확인 및 재시작 설치 버튼 제공
- worktree 생성 UX 개선
  - 새 브랜치 생성 모드와 기존 브랜치 체크아웃 모드 분리
  - branch name 기반 worktree path 자동 추천
- 개발 환경 안정화
  - Electron preload 번들이 dev watch 중 덮어써지지 않도록 수정
  - dev 앱 재시작 watch 범위 축소
