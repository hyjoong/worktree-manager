# v0.1.7

## 주요 변경

- packaged 앱 시작 직후 흰 화면 또는 종료처럼 보일 수 있는 문제를 줄이기 위해 시작 시 자동 업데이트 확인을 비활성화했습니다.
- 업데이트 확인은 우측 `App` 탭 또는 상단 다운로드 아이콘에서 수동으로 실행합니다.

현재 빌드는 아직 Apple Developer ID signing과 notarization을 적용하지 않은 unsigned 빌드입니다.
처음 실행할 때 macOS 보안 경고가 표시될 수 있습니다.
