# v0.1.5

## 주요 변경

- 자동 업데이트 배포 채널을 public release repo로 변경
  - 소스 코드 저장소는 private 유지
  - 앱 업데이트 산출물은 `hyjoong/worktree-manager-releases`에 업로드
  - 앱이 인증 없이 GitHub Releases 피드와 `latest-mac.yml`을 읽을 수 있도록 구성
- private GitHub release feed 오류 안내 개선
  - 긴 GitHub 응답 헤더 대신 짧은 원인 안내를 표시

## 배포 파일

- `Worktree Manager-0.1.5-mac-arm64.dmg`
- `Worktree Manager-0.1.5-mac-arm64.zip`
- `latest-mac.yml`
- `.blockmap` 파일

## 참고

현재 빌드는 아직 Apple Developer ID signing과 notarization을 적용하지 않은 unsigned 빌드입니다.
처음 실행할 때 macOS 보안 경고가 표시될 수 있습니다.
