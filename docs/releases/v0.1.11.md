# v0.1.11

## 주요 변경

- 새 worktree 생성 다이얼로그에서 브랜치명 기반 경로 추천을 제공합니다.
- 프로젝트명 prefix 경로와 짧은 경로를 선택할 수 있어 직접 path를 입력하는 부담을 줄였습니다.
- 기존 브랜치로 worktree를 만들 때 local/remote branch를 검색해서 선택할 수 있습니다.
- `Create & Open` 액션을 추가해 worktree 생성 후 바로 선택한 editor로 열 수 있습니다.
