# v0.1.8

## 주요 변경

- macOS 배포 빌드에 Apple Developer ID signing을 적용했습니다.
- hardened runtime과 Electron용 entitlements를 추가했습니다.
- Apple notarization 기반 배포 흐름을 구성했습니다.
- unsigned 로컬 빌드를 위한 `pnpm dist:mac:unsigned` 스크립트를 추가했습니다.
- README에 signing/notarization 배포 절차와 검증 명령어를 추가했습니다.

## 배포 파일

- `Worktree Manager-0.1.8-mac-arm64.dmg`
- `Worktree Manager-0.1.8-mac-arm64.zip`
- `latest-mac.yml`
- `.blockmap` 파일

## 참고

이번 버전부터 macOS 배포 산출물은 Developer ID 서명과 notarization 검증을 통과한 빌드를 기준으로 배포합니다.
